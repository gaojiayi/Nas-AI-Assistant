# 飞牛OS (FnOS) 部署 iStoreOS 旁路由配置指南

通过在飞牛OS的虚拟机中部署 iStoreOS 作为旁路由（辅助网关），可以接管局域网内部分或全部设备的网络流量，实现更高级的路由和网络管理功能。

## 前期准备

- **飞牛OS (FnOS) 已正常运行**：且设备支持并已开启虚拟机功能。
- **下载 iStoreOS 固件**：前往 iStoreOS 官网，下载适用于 x86 架构的固件包（通常为 .img 或 .img.gz 格式，若是压缩包请先解压出 .img 文件）。
- **网络规划**：
  - 假设主路由 IP 为：`192.168.1.1`
  - 规划 iStoreOS 旁路由 IP 为：`192.168.1.2`（确保此 IP 未被局域网内其他设备占用）

## 步骤一：在飞牛OS中创建虚拟机

1. 登录飞牛OS后台，打开**虚拟机管理器**应用。
2. 点击**新建虚拟机**，选择 Linux 系统类型。
3. **分配资源**：
   - **CPU**：建议分配 2 核以上。
   - **内存**：建议分配 2GB 或以上.
4. **添加虚拟磁盘**：将下载好的 iStoreOS .img 固件作为系统引导盘导入或映射（具体取决于 FnOS 的导入逻辑，可能需要将 img 转换为 qcow2，或直接挂载为 RAW 格式磁盘）。
5. **网络设置（关键）**：
   - 将虚拟机的网络模式设置为**桥接模式 (Bridge)**，并绑定到飞牛OS连接主路由的那个物理网口。
   - 这一步确保 iStoreOS 能够获取与 NAS 同一网段的 IP，并与局域网其他设备互通.
6. **保存配置并启动虚拟机**.

## 步骤二：配置 iStoreOS 网络（终端操作）

1. 虚拟机启动后，通过飞牛OS的 **VNC/控制台** 功能进入 iStoreOS 的命令行界面.
2. 待跑码结束后，按下回车键激活终端.
3. 输入 `quickstart` 命令进入快捷配置向导（iStoreOS 特色功能）。
4. 选择 `1. Change LAN IP` (修改局域网 IP).
5. 输入您规划的旁路由 IP，例如 `192.168.1.2`.
6. 输入子网掩码，通常为 `255.255.255.0`.
7. 确认修改并退出.系统会重启网络服务.

## 步骤三：配置 iStoreOS 旁路由网关

1. 现在可以在局域网内的电脑浏览器中，输入刚刚设置的 IP `192.168.1.2` 登录 iStoreOS 后台（默认密码通常为 `password`）。
2. 进入 **网络 -> 接口 -> LAN**，点击"修改".
3. **IPv4 网关**：填写主路由的 IP（例如 `192.168.1.1`）。
4. **使用自定义的 DNS 服务器**：填写主路由 IP `192.168.1.1`，或填写公共 DNS（如 `223.5.5.5`、`114.114.114.114`）。
5. **DHCP 服务器**：
   - 在"基本设置"中，勾选**"忽略此接口"**（关闭旁路由的 DHCP 功能，防止与主路由的 DHCP 冲突，这是最稳妥的旁路由模式）。
6. 点击**保存并应用**.
7. 此时，iStoreOS 本身已经可以正常联网了.可以在 iStoreOS 的"网络诊断"中 ping 一下百度测试是否通畅.

## 步骤四：将局域网设备接入旁路由

要让局域网内的设备通过 iStoreOS 旁路由上网，有两种方式：

### 方式 A：全局接管（所有设备默认走旁路由）

1. 登录您的主路由管理后台.
2. 找到 **LAN 口设置 / DHCP 服务器设置**.
3. 将**默认网关**修改为 iStoreOS 的 IP（`192.168.1.2`）。
4. 将**首选 DNS**也修改为 iStoreOS 的 IP（`192.168.1.2`）。
5. 保存并重启主路由，局域网内的设备重新连接 Wi-Fi 后，流量就会自动经过旁路由.

### 方式 B：按需接管（仅指定设备走旁路由）

如果您不希望影响家里其他人的网络，可以只在需要的设备上手动配置：

1. 打开您的手机或电脑的 Wi-Fi/网络设置.
2. 将 IP 获取方式从"自动(DHCP)"改为"手动/静态 IP".
3. 填写一个固定的局域网 IP.
4. **路由器/网关**：填写 iStoreOS 的 IP（`192.168.1.2`）。
5. **DNS**：填写 iStoreOS 的 IP（`192.168.1.2`）。
6. 保存后，仅该设备的流量会经过旁路由处理.

## 进阶应用建议

完成旁路由配置后，您可以在 iStoreOS 的"iStore 应用商店"中安装合规的网络管理插件：

- **AdGuard Home**：用于拦截局域网内的恶意网站和广告.
- **WireGuard Server**：搭建安全的私人 VPN 隧道，当您在公司或户外时，可以安全地拨号连接回家中，直接访问局域网内的飞牛OS NAS 数据.
